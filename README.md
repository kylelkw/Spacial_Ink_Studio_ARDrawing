\# Spacial Ink Studio - AR Drawing Application



Meta Quest 3S drawing application built with Unity in 12 hours.



\## Project Info

\- \*\*Platform:\*\* Meta Quest 3S

\- \*\*Engine:\*\* Unity 2022.3 LTS

\- \*\*Development Time:\*\* 12 hours

\- \*\*Date:\*\* January 17, 2026



\## Features (Planned)

\- Wire brush (3D spatial drawing)

\- Flat brush (graffiti-style billboard)

\- 8 color palette

\- Analog size control via joystick

\- Erase and undo functionality

\- Extensible brush system
